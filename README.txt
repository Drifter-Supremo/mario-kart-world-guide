# Mario Kart World - Interactive Unlock & Strategy Guide

## Overview
This is a comprehensive, interactive guide for Mario Kart World that helps players track their progress and find the best strategies for each game mode.

## Features
- **Interactive Progress Tracking**: Check off unlocked items and track collectibles
- **Comprehensive Unlock Guide**: Complete roadmap of all unlockable content
- **Strategy Recommendations**: Best character-kart combinations for each game mode
- **Search Functionality**: Find specific characters, karts, or items quickly
- **Responsive Design**: Works on desktop and mobile devices
- **Dark Mode Friendly**: Optimized for dark theme preferences
- **Offline Functionality**: No internet connection required after download

## Files Included
1. `mario-kart-world-guide.html` - Main HTML file with embedded data
2. `style.css` - CSS stylesheet for styling and responsive design
3. `README.txt` - This documentation file

## How to Use
1. Open `mario-kart-world-guide.html` in any modern web browser
2. Navigate between tabs to access different sections:
   - **Overview**: Dashboard with statistics and quick access
   - **Unlock Road-Map**: Complete list of unlock requirements
   - **Grand Prix**: Best loadouts for each CC class
   - **Knockout Tour**: Survival racing strategies
   - **Battle**: Arena combat tips
   - **Free Roam**: Collectibles and mission tracking
3. Use checkboxes to track your progress (automatically saved)
4. Use the search bar to find specific content
5. Apply filters in the sidebar to narrow down results

## Progress Tracking
Your progress is automatically saved to your browser's local storage. This includes:
- Checked unlock items
- Collectible counts
- Completed missions

## Data Sources
All unlock requirements and strategies are based on official Nintendo sources and verified game guides as of June 14, 2025.

## Technical Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- No internet connection required after initial download

## Updating Data
To update the game data:
1. Open `mario-kart-world-guide.html` in a text editor
2. Locate the `<script type="application/json" id="gameData">` section
3. Modify the JSON data structure as needed
4. Save the file

The JSON structure includes:
- `characters`: Character data with stats and unlock requirements
- `karts`: Kart data with types and unlock methods
- `tracks`: Track information and unlock requirements
- `collectibles`: Collectible totals and descriptions
- `gameModes`: Game mode information and features
